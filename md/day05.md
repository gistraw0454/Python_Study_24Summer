## 형변환
1. 자동 형변환
    - 정수 + 정수 = 정수
    - 정수 + 실수 = 실수
    - 3 + 0.0 = 3.0
    - `print("data4 : %c" %65)`
2. 강제 형변환
    - 자료형(값)
    - int(10.98)==10
    - print(10//3.0)
    - print(float(10)//3)
