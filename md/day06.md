# 6강 - 암호화와 복호화, 입력함수
## 문자 형변환
- `chr(정수)` : 정수를 문자로
- `ord(문자)` : 문자를 정수로
- 'A' -1 이런거 계산하려면 문자를 정수로 바꿔줘야함.
- 아스키코드를 통해 암호화할 수 있다.
- 회원가입 시 사용자의 비밀번호 혹은 개인정보를 암호화할때 아스키 코드를 사용할 수 있다.
```py
#암호화
pw = "a1b2c3"
en_pw = ""
de_pw = ""

for i in pw:
    en_pw+=chr(ord(i) *9)
print("기존 비밀번호 : %s" %pw)
print("암호화된 비밀번호 : {pw}".format(pw=en_pw))


for i in en_pw:
    de_pw += chr(ord(i)//9)
print("복호화된 비밀번호 : {pw}".format(pw=de_pw))
# 기존 비밀번호 : a1b2c3
# 암호화된 비밀번호 : ??????
# 복호화된 비밀번호 : a1b2c3
```
