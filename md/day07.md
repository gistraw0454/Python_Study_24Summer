# 7강 - 연산자1 - 우선순위와 결합성
## 연산자
- 기능이있는 특수문자
### 연산자 우선순위
```
최 최우선 연산자
단 단항 연산자
산 산술 연산자
쉬 쉬프트 연산자
관 관계 연산자
리 논리 연산자
삼 삼항 연산자
대 대입 연산자
```
항상 왼쪽 -> 오른쪽 연산

### 결합성 
- 하나의 수식에 같은 우선순위의 연산자들이 여러개 나올 때 알맞는 방향으로 결합되어 연산되는 성질

> **입력값은 항상 문자열로 들어간다.**

ex) price = `int(input("가격 : "))`


## 조건식
- 결과가 참 또는 거짓 둘 중 하나가 나오는 식
### 파이썬에서 참거짓
- True : 참
- False : 거짓

#### 관계 연산자
- ==, !=, <, >, <=, >=,

#### 논리 연산자
-  and , or

#### 단항 연산자
- not
