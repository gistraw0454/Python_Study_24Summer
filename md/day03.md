# 3강 - 변수의 선언과 자료형

## 자료형 type
- 정수형 int
- 실수형 float 
- 문자열 str
- 논리형 bool

## 변수선언 (메모리 할당)
- `변수명 = 초기값` [초기화] 
- 값을 뭘 넣던간에 넣은 값에 따라 저장공간의 종류인 자료형이 바뀜.

### 변수 선언시 RAM
- x=0 : x라는 이름의 저장공간이 RAM에 할당됨.
- x에 0이라는 초기값이들어감.
### 주소 
- 저장공간이 RAM에 할당되면 고유한 값인 주소값이 부여됨 . (KVN이 알아서 주소를 관리해줌)

### 변수 선언시 주의사항
- 같은 이름으로 선언하면 기존값 바꾸는 일 뿐임. 이딴짓 x
- 되도록 선언부에 한꺼번에 선언한다. (페이지 상단)

### 변수명 주의사항
- 문자부터 시작
- 특수문자 사용 X
    - `_`는 사용가능
- 공백 불가 
    - good_boy : 팟홀 표기법
    - goodBoy : 카멜 표기법
- 되도록 소문자 사용
- 되도록 뜻 있는 단어 선택
