## 함수: 어떠한 영역에도 갇혀있지 않고 선언된 것
    - 이름뒤에 소괄호/단, 키워드()는 함수가 아니다 .
        - if for 이런거
### 함수의 선언
```
def 메소드명(매개변수,..):
    실행문장
    return 리턴값
```
1. 메소드명 : 동사로 작성
2. 외부에서 값을 전달받기 위해 반드시 선언, 생략가능
3. 생략가능
4. 생략가능

### 메소드 선언시 주의사항
- 메소드 내에서 메소드를 웬만하면 선언하지 않고, 메소드를 사용할때에는 보통 메소드 안에서 사용한다
- 매개변수가 없을때 사용시 값을전달할 수 없다
- 리턴값이 있다면 사용하는 부분 통채로를 리턴값으로 보자

### 메소드 선언 순서
1. 메소드 이름 생각
    - def add():
2. 매개변수 생각
    - def add(a,b):
3. 실행문장
    - def add(a,b):
        - result = a+b
4. 리턴값 생각
    - def add(a,b):
        - result = a+b
        - return result

### 메소드의 목적
1. 재사용의 목적(단, 특정성을 부여해서는 안된다)
2. 소스코드 간결화
#### 리턴해야할때
- 메소드 내에서 모든 작업이 끝나지 않을때
- 사용하는 부분에 메소드 내의 작업 결과를 알려줘야할 때
